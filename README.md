# jadecanvas
Jade Canvas Script 

Purpose: Take exams in the format of 

[#]. [Question]
a. [Answer] b. [Answer] 
c. [Answer] d. [Answer] 

and parse text into a list of strings in the order of question, followed by the 4 possible answers. 

Questions and answers are written into an output file in the format of question \n answer \n etc... 

This allows for easy copy and paste into a single column in a spreadsheet where each cell is either a question or answer 

Usage: Convert Word document exams into Canvas exams
